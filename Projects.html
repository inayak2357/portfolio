Computer Science Principals

Project: Scratch Project

Course: CSP (Computers Science Principals)
Duration: August 30, 2018-September 7, 2018
Type of Project: Partner Project

Scratch Program is a visual programming language. We were given an introduction and then our instructions on the coding and animation to make our scratch game/story. Working on the Scratch project, I learned about the logic involved in coding; using blocks, we are able to manipulate sprites and the different elements including background and sounds. I learned too that commenting can help a lot with figuring out which code does what when a piece of code breaks. For someone who had no background in coding, I realized that perseverance is important. Working in pairs, our story/game that we created for the project included a lot of sprites and scene changes and we found it hard to code for each movement. Besides working hard to learn how to code, it was equally important to learn to work and communicate well with a team mate. Here is our documentation that we did as we worked on our project.
